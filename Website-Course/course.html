Bab 1: Struktur Dasar HTML5

[ Area Pemutar Video Tutorial ]

Konsep Utama

Setiap dokumen HTML membutuhkan deklarasi doctype dan tag root html. Berikut adalah contoh kerangka standar yang wajib dipahami oleh setiap pengembang web.

<!DOCTYPE html>
<html lang="id">
<head>
    <title>Halaman Pertama</title>
</head>
<body>
    <h1>Halo Dunia</h1>
</body>
</html>

Latihan Interaktif

Silakan buat struktur HTML di atas pada editor teks Anda masing-masing, lalu ubah teks pada tag <h1> menjadi nama Anda.
